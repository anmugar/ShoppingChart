Web Application Technologies Project

This is a project for the subject Web Application Technologies done in the AGH University.

The funcionalities of the web are providing a list of products from a database for each user.

The username is stored in a database where the information of the user is get for checking if the login is correct. Also a new users can be registered into the system, whose information will be stored as well.

The products are associated to each user and each user can view, add, edit or delete products in the main panel.
